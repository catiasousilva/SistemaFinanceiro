Siga os passos abaixo para montar o ambiente:

1. Configuração do banco de dados

1.1. Configure o ambiente MySql.
1.2. Crie o schema do banco de dados com o nome db_controlefinanceiro.
1.3. Seta o banco de dados db_controlefinanceiro.
1.4. Execute o script contido no arquivo "Script Create Table db_controlefinanceiro.sql" para criar a estrutura do banco.
1.5. Execute o script "Script Insert Usuário.sql" para criar o acesso do Sistema Financeiro. Perfil Gerente tem acesso a todos módulos do sistema.

---------------------------------------------------------------------------------------------------------------------------------------------------

2. Configuração da aplicação

2.1. Copie o diretório "SistemaFinanceiro" para a raiz do seu sistema operacional ou para o endereço que desejar.
2.2. Execute a solution SistemaFinanceiro.sln
2.3. Compile o projeto.
